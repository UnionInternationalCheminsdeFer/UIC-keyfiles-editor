Import and Export of Keys

The editor menu and context menu provide functions to import and export the content of keys. Keys can be imported from and exported to .pem files either as certificates or as public keys.

.pem is a simple file format to include public keys or certificates. The file starts with an text indicator e.g. "-----BEGINN CERTIFICATE" followed by the base-64 encoded binary key or certificate. It ends with an end indicator i.e. ------END OF CERTIFICATE.

functiondescription
add certificate from pem file Add the certificate from a certificate pem formatted file.
add public key from pem file Add a public key from a pem formatted file. The pem file can be a public key pem file or a certificate pem file.
save to pem fileSave the public key or certificate to a pem formatted file
display public key contentdisplays the content of a public key or certificate
