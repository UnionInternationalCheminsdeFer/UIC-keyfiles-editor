Menus

The application provides the following menus:

Functions to create, open and close key files in the file menu. Several files can be open at the same time.

File menu



Edit menu

Functions to edit key files in the edit menu. These include cut, copy and paste and undo or redo. Cut copy and paste can be applied between different files.

    functiondescription
    cutcuts the selected element to the clipboard
    copycopy the selected element to the clipboard
    pastepaste the clipboard content in the selected item
    undoundo the last change on the selected file
    redoredo the last undone change on he selected file
    deletedelete the selected items


Editor menu

Special functions to edit key files in the editor menu. These include creating of new keys in the file and special functions to import and export the key to other formats and to display the key content. Validation of the content is available there too.



    functiondescription
    create new siblingCreates a new sibling of the selected item - only available on lists
    create new childCreates a new child item too the selected item
    Validate / AuswertenValidates the selected item
    save test bar codeSave an included test bar code as image file.
    Available only if the key contains a test bar code.
    display test bar codeDisplay an included test bar code.
    Available only if the key contains a test bar code.
    add test bar codeAdds a test bar code from an image file (jpg, png). An existing test bar code is replaced. (jpg, png)
    add certificate from pem file Adds the certificate from a certificate pem formatted file.
    add public key from pem file Adds a public key from a pem formatted file. The pem file can be a public key pem file or a certificate pem file.
    save public key to pem fileSaves the public key to a pem formatted file containing a public key.
    save certificate to pem fileSaves the certificate to a pem formatted file.
    Available only in case the xml file contains a certificate.
    save all keys to pem filesSaves all keys to pem files using the issuer code and the id in the file name.
    save to pem fileSaves a public key to a pem file. In case the public key is included in a CERTIFICATE the pem will include the certificate.
    display public key contentDisplays the content of a public key or certificate
