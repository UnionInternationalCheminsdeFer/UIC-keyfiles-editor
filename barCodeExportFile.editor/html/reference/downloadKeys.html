The application provides the option to download keys from the UIC PKMW site:

The function to download the keys from the PKMW site of UIC is in File menu. It will ask for a local file name before starting the download.









The URL to the PKMW download can be changed in the UIC preferences located in the preferences menu under the Help menu.
To restore he default URL click the restore defaults button on the UIC preferences page.





In case you access the download within a company network you might need to use a proxy server. You can add a proxy server in the network preferences.
Please note, that the UIC PKMW is using https, so the proxy needs to be configured and activated for https!
