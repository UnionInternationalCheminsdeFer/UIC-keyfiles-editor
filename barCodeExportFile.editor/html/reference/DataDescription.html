Description of data elements

The download file for keys contains a list of keys. Each key has the following attributes:

    Tag NameValue
    issuerName Name of the company which deliver the key
    issuerCode Railways RICS code printed on the ticket. Code needs to be in the list
    versionTypeType of the bar code used
    • UIC 918-2
    • UIC 918-3
    • UIC 918-2PDF417
    • NonUIC*

    (* bilateral bar code definition must start with “NonUIC”)
    barcodeVersion Subversion within the versionType 2,3
    id ID of the key corresponding to the ID of signature used in the bar code
    signatureAlgorithmName of the algorithm used for the signature. The name also indicates whether the signature is included as standard asn.1 encoded structure or as two separate signature parameters:
    • SHA256-DSA2048-NoASN1 (signature is included in the bar code as 2 numbers without wrapping standard asn.1 structure)
    • SHA224-DSA2048-NoASN1 (signature is included in the bar code as 2 numbers without wrapping standard asn.1 structure)
    • SHA224-DSA2048-ASN1
    • SHA160-DSA1024-ASN1
    • SHA160-DSA512-ASN1
    • SHA256withDSA(2048,256)
    • SHA224withDSA(2048,224)
    • SHA1withDSA(1024,160)
    • SHA1withDSA(512,160)
    • SHA256withECDSA-P256 (elliptic curve signature)
    • NonUIC
    publicKeyTypeType of the public key data included:

    It is recommended to use certificates only.

    • PUBLIC KEY
    • CERTIFICATE
    publicKeyThe public key base64 encoded either as the public key or included in a certificate
    startDateValidity: start date on the train (IRT) or begin of validity of the ticket (NRT)
    endDateValidity: end date on the train (IRT) or end of validity of the ticket (NRT)
    lastDayOfSale Last day where the key has been used to sign bar codes. This date can be used in ticket validation.
    barcodeXsd bar code description (deprecated)
    allowedProductOwnerCodesCode and Name of the railway companies which allow the key for be used for their products(trains) with the two following sub elements
    productOwnerCode Railways RICS code to control the ticket. Code needs to be in the list on the official RICS code list
    productOwnerName Railways name
    keyForged Boolean to indicate forgery
    commentForEncryptionTypeComment on the encryption method
    testbarcodeImage of A test bar code
    imagetypeType of the image (png, jpg or bmp) of a test bar code
    imagedataBase64 encoded image data of a test bar code
