Changes

The functions to export a certificate or public key as a pem formated file have been extended. Function to export the security feature as a per formatted certificate and as a pem formatted public key have been added. The export functions now check whether the security feature included in the xml file is realy a public key or certificate and do not rely on the description in the xml file.

Getting Started

This application provides to function to create, modify and view public key exchange xml files structured according to the UIC leaflet 918.2. These files can be down loaded from the UIC web page for key exchange and can be submitted to UIC for publication.

Application parts

The application shows three windows: The main window to display the open key files, the Outline View showing the content of the selected key file and the property view showing the content of the elements of the selected part of a key file.





Menus

The menu bar of the application provides access to all functions.

  • Functions to create, open and close key files in the file menu. Several files can be open at the same time.




  • Functions to edit key files in the edit menu. These include cut, copy and paste and undo or redo. Cut copy and paste can be applied between different files.




  • Special functions to edit key files in the editor menu. These include creating of new keys in the file and special functions to import and export the key to other formats and to display the key content. Validation of the content is available there too.




  • Functions are also available in the context menu to the main and the outline view.




  • The about menu shows the license agreements of the software. By using the software you agreed to the conditions.
