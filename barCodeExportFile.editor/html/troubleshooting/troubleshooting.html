Troubleshooting

Network problems


I cannot download the keys file.The connection to the UIC download site might be restricted within your company network. In this you must use a proxy server within your company network. Please ask your local it staff for the address of the proxy server. The proxy server address can be added in the preferences page for network settings. The proxy server must be set for the https protocol.
I have configured the proxy server of my company but i receive a error indication the download is forbidden. Depending on your companies security policy it still might be forbidden to access the web site via an application. In this case you would receive an error reply indicating that the access is forbidden. You then need to download the keys file via your web browser to your disc and open it via the file dialog.
